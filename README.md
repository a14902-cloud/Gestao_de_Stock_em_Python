# Gestão_de_Stock_em_Python
# 📦 Gestão de Stock de Materiais Escolares  
Programa desenvolvido em Python

Este projeto corresponde a um exercício cujo objetivo é **desenvolver um programa em Python que utiliza funções para gerir o stock de materiais escolares de uma escola**.  
O sistema permite o **registo**, **consulta** e **atualização** do stock de forma eficiente, recorrendo a um menu interativo em consola.

---

## 🎯 Enunciado do Exercício

Desenvolver um programa em Python que utiliza funções para gerir o stock de materiais escolares de uma escola.  
O programa deverá permitir:

- Registar novos materiais  
- Consultar o stock existente  
- Atualizar quantidades (adicionar ou remover)  
- Exibir o estado geral do stock  

---

## 🧠 Funcionalidades Implementadas

| Função | Descrição |
|-------|-----------|
| `adicionar_material()` | Regista um novo material no stock |
| `consultar_stock()` | Consulta a quantidade disponível de um material |
| `atualizar_stock()` | Adiciona ou remove unidades de um material |
| `exibir_stock()` | Mostra todos os materiais e respetivas quantidades |
| `main()` | Controla o menu e o fluxo do programa |
